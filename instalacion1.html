Instala Windows

Con esta guia de pasos puedes instalar Windows 10 en tu computadora o podrias instalar en un virtualizador para ralizar pruebas.

En mi caso usaré el virtulizador de VirtualBox, puedes decargar aqui para que puedas seguir los mismos pasos sin riesgos de dañar tu sistema operativo actual

Para poder usar instalar un sistema operativo en un virtualizador primero crear la maquina virtual dando caranteristicas que tendri una computadora como el tamaño del disco duro y los nucleos que se puede compartir de una manera no afecte a tu computadora.

Una vez creado comenzamos a encender la maquina virtulal. Una vez iniciado el programa te pedirá el disco de inicio donde debemos colocar nuestro Archivo de Imagen de Disco y colocamos Iniciar que lo puedes descargar desde la paguina oficial de Windows.

Presionamos sobre la version de 64 bits para poder comenzar la Instalación de Windows 10

En esta apartado colocamos el idioma de preferencia del programa de instalación y presionamos en Siguiente.

Presionamos instalar ahora para comenzar el programa de instalación de Windows y esperamos a que el programa de instalación se inicie.

Una vez iniciado el programa de instalación nos pedirá clave del producto para omitir presionar donde dice No tengo una clave de producto, para continuar con la instalacion sin tener una clave

En este apartado seleccionamos el tipo del sistema operativo que más nos interece en mi caso Windows 10 Home y presinamos en Siguiente.

Ahora aceptamos todos los términos de licencia que tiene Windows y presionamos siguiente.

En esta parte es necesario elejir la instalación de tipo personalizada para realizar las particiones del disco duro.

En aquí podemos hacer las particiones donde podemos colocar varias unidades presionando en nuevo y asignar la cantidad de espacio tendrá cada uno. Es recomendable que cuente con un maximo de 2 o 3 particiones para tener una mejor organización.

Teniendo en cuenta que el sistema operativo se instalara en la primera partición, presionamos en siguiente para que empieze a crear las particiones y se instale windows.

Ahora solo falta esperar para que se instale Windows 10, esto puede demorar bastante.

Una vez terminada la instalación debemos esperar a que se inicie por completo.

En este apartado utilizaremos la región de México aunque no sea tu pais de origen y presionamos sí.

En la distribución del teclado dejamos en latinoamericano y presionamos sí.

Ahora ingresamos nuestra cuenta de Microsoft para poder iniciar la cesion o podemos crear una en el apartado de abajo.

iniciada la sesión ya podemos utilizar el sistema operativo de Windows 10.
